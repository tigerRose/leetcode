## 找出数组中四个数相加最接近某个数的结果

### 问题描述

找出数组中四个数相加最接近某个数的结果

**input** [1,0,-1,0,-2,2]  0

**output** [[-1,1,0,0], [-2,2,0,0], [-2,2,-1,1]]

### 个人思路

找规律：

两个数的和为某个数，在数组首尾各维持一个指针，只需遍历一遍，时间复杂度O(n)

三个数的和为某个数，循环数组元素，继续维持两个指针，需遍历两层，时间复杂度O(n^2)

因此推断，四个数的话，首先两层循环来遍历元素，再维持两个指针，时间复杂度O(n^3)


看了答案，可以使用递归写一个N个数的计算的函数，时间复杂度O(n^(N-1))
